CSS Fundamentals Lab

This project demonstrates CSS selectors, typography,
background styling, image effects, and visual design.

## Complete Lab (Task 4–13) mapping for grading

- `exercises/complete-lab.html` + `styles/lab.css`
- Task 4: image gallery styling with `.portfolio`, `.overlay`, hover transitions, filters
- Task 5: buttons styled with `.btn`, `.btn-primary`, `.btn-secondary`, `.btn-round`, focus/disabled states
- Task 6: form layout with `.form-card`, `input:focus`, `.error`, `.success-msg`, custom focuses
- Task 7: card components with `.card`, `.card-grid`, `.card:hover`, shadows and transforms
- Task 8: navigation with `.navbar` (horizontal + vertical), active/hover states, responsive menu
- Task 9: transform gallery using `.transform-card`, `.transform-box`, hover `rotate`, `scale`, `translate`, `skew`
- Task 10: advanced effects `.gradient-text`, `.clip-text`, custom checkboxes/radios, `.loader`, `.complex-shadow`
- Task 11: portfolio section `.portfolio-page`, `.portfolio-card`, spacing and responsive layout
- Task 12: table styling `.custom-table` with zebra stripes, hover row effect
- Task 13: UI enhancement `.site-enhanced` with gradient, spacing, and polished presentation
